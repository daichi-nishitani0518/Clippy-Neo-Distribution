# INSTALL

## インストール手順

1. Releases ページから最新の DMG ファイルをダウンロードします
2. ダウンロードした DMG ファイルを開きます
3. `Clippy Neo.app` を `Applications` フォルダへドラッグします
4. `Applications` フォルダから `Clippy Neo` を起動します

## 初回起動時の注意

初回起動時は、macOS の設定やセキュリティ警告により、そのまま開けない場合があります。
その場合は、以下の手順を試してください。

## 「開発元を確認できません」と表示された場合

1. `Clippy Neo.app` を閉じます
2. Finder で `Applications` フォルダを開きます
3. `Clippy Neo.app` を右クリックします
4. `開く` を選択します
5. 確認ダイアログが表示されたら、もう一度 `開く` を選択します

それでも開けない場合は、以下も確認してください。

1. `システム設定` を開きます
2. `プライバシーとセキュリティ` を開きます
3. 画面下部に `Clippy Neo` に関する警告が表示されていれば `このまま開く` を選択します

## 必要な権限について

Clippy Neo の機能を利用するため、以下の権限が必要になる場合があります。

### アクセシビリティ

貼り付け操作の補助や、一部機能の動作に必要です。

設定手順:

1. `システム設定` を開きます
2. `プライバシーとセキュリティ` を開きます
3. `アクセシビリティ` を選択します
4. `Clippy Neo` を許可します

## アップデート方法

1. Releases ページから新しいバージョンの DMG をダウンロードします
2. 新しい `Clippy Neo.app` を `Applications` フォルダに上書きします

### v0.1.2 のデータ移行に関する注意

`v0.1.2` 以降、スニペットのインポート / エクスポート形式は JSON ではなく XML です。
バックアップや別環境への移行を行う場合は、設定画面から XML ファイルを出力して利用してください。

### アップデート後に貼り付けが動かない場合

現在の配布版は ad-hoc 署名です。
macOS では、アプリを新しいビルドへ差し替えたあとにアクセシビリティ権限の再付与が必要になることがあります。

その場合は、以下の手順を実施してください。

1. `Clippy Neo` を終了します
2. `システム設定` を開きます
3. `プライバシーとセキュリティ` → `アクセシビリティ` を開きます
4. 一覧にある `Clippy Neo` をいったん削除します
5. `+` ボタンから `Applications/Clippy Neo.app` を再追加します
6. `Clippy Neo` を起動し直します

## アンインストール方法

1. `Applications` フォルダから `Clippy Neo.app` を削除します
2. 必要に応じて関連データを手動で削除してください
