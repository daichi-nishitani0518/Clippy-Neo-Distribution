# Clippy Neo

AI機能やOCRにも対応した、多機能な macOS 向けクリップボードアプリです。

## できること

- 複数のコピー履歴を保存し、必要なものを選んで貼り付けできます
- テキストだけでなく画像のコピー履歴も扱えます
- よく使う定型文をスニペットとして保存できます
- AI連携により、文章の校正や Markdown 形式での文章修正が可能です
- AI機能はローカルLLMとクラウドLLMの両方に対応しています
- OCR機能により、画像内の文字を抽出できます
- 翻訳機能により、日本語と英語の翻訳に対応しています

## AI機能について

Clippy Neo の AI機能は、ローカル環境とクラウド環境の両方で利用できます。

設定画面から、AI プロバイダー、エンドポイント、APIキー、モデルを設定してください。

対応している主なプロバイダーは以下のとおりです。

- Ollama Local
- Ollama Cloud
- OpenAI
- Anthropic
- Custom endpoint

ローカルで利用する場合は、Ollama Local を選択し、必要なモデルをあらかじめインストールしてお使いください。
クラウドで利用する場合は、利用するプロバイダーに応じた APIキーの設定が必要です。
