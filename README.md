# Clippy Neo

AI機能やOCRにも対応した、多機能な macOS 向けクリップボードアプリです。

## できること

- 複数のコピー履歴を保存し、必要なものを選んで貼り付けできます
- テキストだけでなく画像のコピー履歴も扱えます
- よく使う定型文をスニペットとして保存できます
- AI連携により、文章の校正や Markdown 形式での文章修正が可能です
- OCR機能により、画像内の文字を抽出できます
- 翻訳機能により、日本語と英語の翻訳に対応しています

## 対応環境

- macOS
- Apple Silicon 搭載 Mac
