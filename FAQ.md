# FAQ

### 何を保存できますか？

コピーできるものであれば、テキストと画像の両方に対応しています。

### 画像も扱えますか？

はい。画像のコピー履歴にも対応しています。

### 履歴は何件まで保存されますか？

最大 50 件まで保存されます。

### 呼び出しショートカットは何ですか？

`Command + Shift + V` です。

### スニペットのインポート / エクスポート形式は何ですか？

`v0.1.2` 以降は XML 形式です。設定画面から XML ファイルとしてエクスポートし、そのままインポートできます。

### 日本語 IME でフォルダ名を入力すると確定 Enter で即作成されますか？

いいえ。`v0.1.2` 以降は、IME の変換確定 Enter では作成 / リネームを確定せず、その後の通常 Enter で確定する挙動に修正されています。

### 権限の許可はなぜ必要ですか？

貼り付け操作の補助や、一部機能を正常に動作させるために必要です。
詳しくは [INSTALL.md](./INSTALL.md) を参照してください。
